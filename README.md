# PedalboardFactory

Web app for planning guitar pedalboard layouts. Place boards and pedals on a canvas to check fit and arrangement before building.

## Features

- Boards and devices with real-world dimensions (mm)
- Drag, rotate, arrange; pinch-to-zoom on touch (separate from drag)
- Zoom and pan; optional grid in mm or inches
- Ruler tools: rectangle for distance, polyline for cable length
- Export to JSON, load from file (saves current layout only)
- Confirmation before clearing or deleting
- Image assets for common brands (Boss, MXR, Dunlop, Pedaltrain, etc.)

## Getting Started

**Prerequisites:** Node.js v18+, npm v9+

```bash
git clone https://github.com/yourusername/pedal.git
cd pedal
npm install
npm run dev
```

Open `http://localhost:5173`.

## Tech Stack

React 18, Vite, TypeScript, CSS3.
